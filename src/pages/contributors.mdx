# 贡献者

汪汪队教程站第一版先采用 GitHub Pull Request 协作。等仓库公开并开始接受投稿后，这里会整理维护者、教程作者和长期贡献者。

## 当前角色

| 角色 | 负责内容 |
| --- | --- |
| 站点维护者 | 站点结构、发布流程、内容审核 |
| 教程作者 | 新增原创教程、修正已有教程 |
| 审核者 | 检查步骤、截图、链接和构建结果 |

## 如何成为贡献者

请先阅读 [投稿指南](/docs/contribute/write-a-tutorial)。第一篇教程合并后，你就会出现在贡献记录里。
