# 教程模板

新增教程时可以复制下面模板。

````md
---
title: 教程标题
description: 一句话说明读者完成后会得到什么
tags: [topic, tool]
---

# 教程标题

作者：Your Name
更新日期：2026-06-22

## 适合谁

写清楚读者需要具备什么背景。

## 准备工作

- 工具版本：
- 账号或权限：
- 示例项目：

## 操作步骤

### 第一步

说明要做什么，为什么做，完成后应该看到什么。

```bash
command example
```

### 第二步

继续写可复现步骤。

## 验证结果

说明如何确认教程已经完成。

## 常见问题

### 问题标题

原因和解决方式。

## 下一步

可以继续阅读或尝试什么。

## 参考资料

- [资料标题](https://example.com)
````
